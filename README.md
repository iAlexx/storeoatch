# دكانك (SyriaStore Builder) — Unicorn SaaS Investor Platform

Apple Vision Pro inspired spatial SaaS investor platform built with Next.js, mobile-first UX, interactive motion, and realistic SaaS economics.

## What is included

- 3D-inspired glassmorphism visual system with animated cosmic background.
- Full investor deck sections (problem, market, solution, GTM, risks, moat, roadmap).
- Realistic SaaS financial engine for MRR/ARR, margin, LTV/CAC, break-even.
- Mobile-first responsive experience for iPhone/Android.

## Tech Stack

- Next.js 14 + React 18 + TypeScript
- Framer Motion + GSAP + Three.js dependency for 3D expansion
- Ready for Vercel deployment

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run start
```

## Deployment (Vercel)

1. Push repository to GitHub.
2. Import project on Vercel.
3. Build command: `npm run build`
4. Output: Next.js default.
